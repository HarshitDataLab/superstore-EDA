# 📊 Superstore Sales Analysis using Python

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on the Superstore dataset using Python. The goal is to uncover sales trends, profit patterns, customer behavior, and regional performance through data cleaning, visualization, and statistical analysis.

## 🎯 Objectives

* Analyze overall sales and profit performance.
* Identify top-performing products and categories.
* Discover regional and segment-wise sales trends.
* Explore shipping and customer-related insights.
* Create visualizations to support business decisions.

## 🛠️ Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## 📂 Dataset Information

The dataset contains information about:

* Orders
* Customers
* Products
* Sales
* Profit
* Shipping Details
* Regions and States

## 📈 Analysis Performed

### Data Cleaning

* Checked missing values
* Removed duplicates
* Corrected data types

### Exploratory Data Analysis

* Sales and Profit Distribution
* Category & Sub-Category Performance
* Regional Analysis
* Segment Analysis
* Monthly Sales Trends
* Top and Bottom Performing Products

### Visualizations

* Bar Charts
* Line Charts
* Histograms
* Heatmaps
* Boxplots

## 🔍 Key Insights

* Technology category generated the highest sales.
* Certain states contributed significantly to overall profit.
* Some products achieved high sales but low profitability.
* Sales showed seasonal fluctuations across months.


## 🚀 How to Run

1. Clone the repository.
2. Install required libraries:

   ```bash
   pip install pandas numpy matplotlib seaborn
   ```
3. Open Jupyter Notebook:

   ```bash
   jupyter notebook
   ```
4. Run `superstore_eda.ipynb`.

## 📌 Conclusion

This analysis provides valuable business insights into sales performance, profitability, and customer trends. The findings can help organizations make data-driven decisions and improve overall business strategy.
